Protect Your Network with AI

Our system uses advanced deep learning autoencoders to detect anomalies, suspicious login attempts, and port scans in real-time.

🔍 Network Anomaly Detection

Uses TensorFlow autoencoder trained on normal traffic patterns to identify deviations that indicate potential threats.

🔐 Login Protection

Detects brute-force attacks and suspicious login patterns to prevent unauthorized access to your systems.

🌐 Port Scan Detection

Monitors network traffic to identify port scanning activities that precede many cyber attacks.

⚡ Automated Response

Simulated automated remediation actions including IP blocking, account locking, and process termination.

Getting Started

  1. Train the model: cd backend && python model/train_autoencoder.py
  2. Install dependencies: pip install -r requirements.txt
  3. Run the server: python app.py
  4. Access dashboard at: http://localhost:5000/dashboard
  5. Default credentials: admin / admin123
